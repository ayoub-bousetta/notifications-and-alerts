Easy Alerts

Notifications & Alerts

Basic use case 


<div class="alerts warns ">
        <span>
            <i class="material-icons">error_outline</i>
        </span>
         <div class="message">
            <b>Warning!</b>
            <p>Lorem ipsum dolor sit amet consectetur adipisicing elit. <a href="#">Click here</a></p>
        </div>
            <i class="material-icons">clear</i>
</div>
